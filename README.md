# react-musicplayer
An online musicplayer developed with react.  
本项目是基于react搭建的一个前端音乐播放器，初衷是用来练手，因此代码有的地方不甚合理，目前只适配了PC端，仍在不断优化中。  
播放器音源暂时取自于网易云音乐的歌单
## 技术栈
react+webpack+redux/react-redux
